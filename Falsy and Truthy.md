# Falsy and Truthy

Ref.:

- https://developer.mozilla.org/en-US/docs/Glossary/Falsy
- https://developer.mozilla.org/en-US/docs/Glossary/Truthy

O valor é considerado falso quando:

```
false
0
-0
0n
""
''
``
null
undefined
NaA
```

O valor é considerado verdadeiro quando:

```
true
{}
[]
1
1.2
"0"
"false"
-1
-1.2
Infinity
-Infinity
```
